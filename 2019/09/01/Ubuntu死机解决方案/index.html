Ubuntu死机解决方案

今天更新软件时又玄学死机了， Win的话就是直接重启(dog), 但Linux就能用些操作抢救一下

下面记录下解决方案

0x00:

Ctrl + Alt + F2进入终端

最方便简单的是用魔法键 Alt + SysRq + r, e, i, s, u, b， 详情可参照维基百科

SysRq用于发送系统请求，将输入直接交于Liunx系统内核处理

r: unRaw 将键盘从原始模式变为XLATE模式

e:tErminate 将SIGTERM发送给所有进程，请求终止

i: kill 将SIGKILL发送给所有进程，强制停止

s: Sync 将数据同步到磁盘

u: Unmount 将磁盘强制挂载为只读

b: reBoot 重启

0x01:

进入终端后，用top命令查找可能假死的进程，用kill命令结束进程。Ctrl + Alt + F1返回开启的图形桌面

0x02:

进入终端后根据图形界面使用
或者

注销重新登录图形界面
